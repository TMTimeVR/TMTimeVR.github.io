Privacy Policy for Monkey bot

Introduction

This Privacy Policy explains how Monkey bot (the "Bot") handles user information and outlines what data is collected, how it is used, and the responsibilities of users regarding their privacy when interacting with the Bot on Discord. By using Monkey bot, you agree to the collection and use of information as described in this policy.

Information Collected by Monkey bot

User ID Verification

  • When users trigger certain commands, the Bot may check the User ID of the user executing the command. This is done to verify permissions, ensure command functionality, and provide personalized responses.
  • No Storage: The Bot does not store User IDs or any other personal information on its servers. The User ID is only checked in real-time during command execution and is not retained afterward.

DSharpPlus Data Handling

Monkey bot is built using the DSharpPlus library. DSharpPlus may handle the following data temporarily to facilitate communication between Discord's servers and the Bot:

  • Message content: When a command is triggered, the content of the message may be processed to understand and execute the command.
  • Guild information: Information about the server (guild) where the command was triggered may be processed to ensure proper functionality.
  • User details: Basic user details, such as username and avatar, may be accessed to provide a personalized experience.

Temporary Storage: When the users trigger any command, the Bot displays in what guild/server and who triggered the command until the Bot gets restarted, updated, crashed or shutdown. Any data handled by DSharpPlus is only stored temporarily in memory and is not retained by the Bot or its servers once the command execution is complete.

Data Storage

  • Persistent Storage: The Bot stores personal data only when blocking access to a user due to breaking the rules by blocking their user id. The bot does not store messages, or any other information, on its servers. Almost all processing is done in real-time.

Third-Party Services

The Bot may use Discord's APIs and services to function. Users are advised to review Discord's Privacy Policy to understand how Discord collects, uses, and stores user data. The Discord Privacy Policy can be found at Discord Privacy Policy.

User Responsibilities

  • By using Monkey bot, users acknowledge that they have reviewed and understood this Privacy Policy and agree to the collection and use of information as described.
  • Users should regularly review the privacy policies of any third-party services, such as Discord, that they interact with when using the Bot.

Changes to This Privacy Policy

The Bot reserves the right to update this Privacy Policy at any time. Users will be notified of any significant changes, and continued use of the Bot after such changes will constitute acceptance of the new policy.
